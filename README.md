# Ejercicios JavaScript - SENA ADSO

Repositorio de ejercicios prácticos de JavaScript desarrollados como parte de la guía de aprendizaje de Pasarelas de Pago con Node.js.

## Archivos

- **Ejericio1.js** - Ejercicio 1
- **Ejercicio2.js** - Ejercicio 2
- **Ejercicio3.js** - Ejercicio 3
- **Ejercicio5.js** - Ejercicio 5 Hecho por nosotros animales
- **Convertidor.js** - Conversor de grados Celsius a Fahrenheit
- **Usuario.js** - Clase Usuario con validación de contraseña mediante expresiones regulares
- **ofertasProductos.js** - Función flecha que filtra un arreglo de productos y devuelve únicamente los que tienen la propiedad

## Tecnologías

- JavaScript
- Node.js v24.16.0
